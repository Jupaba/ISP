Irregular Strip Packing Problem (ISP)

Por: Juan-Pablo Andrés Barahona Aravena

ISP

Irregular Strip packing Problem proviene de los problemas de cortes y empaque (que en ingles se define como Cutting and Pucking Problem) de piezas en n dimensiones de diferentes formas, ya sean regulares o irregulares, que deben ser almacenados en recipientes o contenedores con el objetivo de ahorrar la mayor cantidad de espacio posible pero que posee diversas variantes y causantes que dificultan este problema. Sin embargo, Irregular Strip Packing Problem se caracteriza del resto de los problemas dado que se insertan figuras irregulares dentro de un contenedor particularmente rectangular cuyo largo es definido, pero su ancho varía dependiedo de las formas en como las piezas se van insertando en él y acomodandose a su dimensión.

Este proyecto tiende a implementar el problema de Irregular Strip Packing problem con figuras en dos dimensiones y que la idea de la creación de este módulo es documentar paso a paso, cambios, mejoras, herramientas implementadas en el desarrollo de este proyecto que será realizado en el lenguaje de C++ y visualizado en HTML utilizando el desarrollo web de PHP cuyo link es Link. Se desarrollarán tres Heurísticas en C++, que permiten la ejecución de este problema y principalmente basado en las tesis y otras referencias de nombre1 nombre2 nombre3 que definen (1) No-Fit Polygon (2) No-Fit Rectangle y el más importante (3) Bottom Left.

El objetivo principal de este proyecto es realizar un servicio de aplicación web con la intensión de resolver problemas de inserción de figuras irregulares en dos dimensiones dentro de un contenedor y entregando resultados ideales para su uso en fábricas de manufacutiación de prendas, metales, zapato, etc, que depeden de una lámina como materia prima.
