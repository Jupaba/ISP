17/02/2015

  • 1.1. La idea de realizar un explorador de archivos de instancias disponibles para el usuario, es simplemente para saber que archivos son capaces de cargar y estar listos para ser cargados en el software.
    Se implementa la biblioteca disponible en boost FileSystem, siendo un código capaz de leer y buscar archivos entre directorios indicando el cámino necesario para llegar a él y poder tener acceso desde el software. El código que permite indicar que archivos se encuentran cargados en los directorios es el siguiente.

21/02/2015

  • Para leer los archivos XML, se utilizó la libreria tinyxml2 que permite facilmente parsear datos que provienen de cualquier archivo XML y ser procesados solo en C++. Dadas a las instancias rescatadas en EURO, todas contienen una estructura estandar con el objetivo de reducir lineas de código en las lecturas de dichos archivos y así facilmente lograr encapsular los polígonos de las distintas problematicas presentadas por los diferentes autores.

    La estructura de lectura es encontrada en ReadParameters.cc cuyo archivo de cabezera es ReadParameters.h.
